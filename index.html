Как замерить

01

Целиком на откос

Если вверху окна есть глухое окошко — замер берётся по всему откосу. Жалюзи устанавливаются внутри откоса. Минимальный зазор сверху: 8 см.

✓ Глухое окошко вверху — можно ставить внутри откоса
02

По каждой створке

Каждая створка измеряется отдельно. Подходит для плиссе. Минимальная площадь за одну створку — 1 м². Если меньше — округляется до 1 м².

!

Нет глухого окошка вверху?

Если вверху рамы нет неоткрывающегося окошка, жалюзи внутри откоса заблокируют открывающуюся часть окна.

В таком случае у вас два варианта:

📐 Вариант 1 — Снаружи откоса
Жалюзи крепятся поверх рамы. Замер по откосу, затем добавьте:
+10 см слева · +10 см справа · +15 см сверху
🪟 Вариант 2 — По каждой створке
Жалюзи крепятся на каждую створку отдельно. Замерьте ширину и высоту каждой части окна.
